# Lighting_visualizer
Adopting the concepts of lighting calculation at different levels of the graphics pipeline (polygon, theme, pixel) is the basis of computer graphics. The paper proposes the development of a web-based application that allows users to visually and interactively explore the differences between Flat, Gouraud and Phong of shading models on 3D objects.
Techonologies: GSLS,WebGL,Three.js
